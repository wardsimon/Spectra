Installation

To use spec1d you need to:
  • Use MATLAB 5.1 or higher
  • Download latest version of  MFIT/MVIEW load routines from http://www.ill.fr/tas/matlab/ftp/load.tar.gz
  • Download latest version of  MFIT/MVIEW func routines from http://www.ill.fr/tas/matlab/ftp/funcs.tar.gz
  • Obtain the latest spec1d files from
You should create a directory called 'spec1d' in a directory which should be entered in your matlab path. Below this directory create
another directory called @spec1d. Place the files formatpars.m, loads.m, specdfdp.m, and speclsqr.m in the spec1d directory, and
all other files in @spec1d.
